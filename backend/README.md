# Gestion Pédagogique

Application de gestion académique complète avec FastAPI et MongoDB. Permet la gestion des formateurs, promotions, étudiants, espaces pédagogiques, travaux, livraisons et évaluations.

## Comptes par défaut

- **Directeur** : 
  - Email : `directeur@ecole.com`
  - Mot de passe : `MotDePasse123`

Les formateurs et étudiants sont créés par le directeur, avec mots de passe générés automatiquement.
